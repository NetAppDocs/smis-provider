= What's new in 5.2.7
:icons: font
:imagesdir: ./media/

* There is no upgrade path available for SMI-S Provider 5.2.7.
* You must deploy SMI-S Provider 5.2.7 as a new installation.
* SnapMirror management with Azure Site Recovery is not supported in this version of SMI-S provider.
* SMI-S Provider 5.2.7 is the only supported version of NetApp SMI-S Provider.

*For Windows users*

* This release is not compatible with Windows Server 2012 or System Center Virtual Machine Manager (SCVMM) 2012.
* SMI-S Provider 5.2.7 supports the following operating systems:
