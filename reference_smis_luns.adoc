= smis luns
:icons: font
:imagesdir: ./media/

[.lead]
The `smis luns` command displays LUN information for storage systems.

== Syntax

----

        smis luns
 \[-t \{http | https\}\]
----

== Location

`C:\Program Files (x86)\NetApp\smis\pegasus\bin`

== Privilege level

A user with a valid user name and password

== Parameters

* *[`-t` {`http | https`}]*
+
Protocol to be used: HTTPS (default) or HTTP
